# Living Room Pranks

Place your PNG files for living room pranks in this folder.

## Expected Files:
- sharpie-face.png - Sharpie marks on family photos
- toilet-paper.png - Toilet paper streamers everywhere  
- food-mess.png - Snack crumbs scattered on the couch

## File Naming Convention:
Use the exact names above for the pranks to work properly.
