## Adding Prank Images

To add your own prank images:
1. Navigate to the appropriate room folder in `pranks/`
2. Add PNG files with these exact names:
   - `sharpie-face.png`
   - `toilet-paper.png` 
   - `food-mess.png`
3. The images will automatically appear in the game!
